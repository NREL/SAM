Graphs

The Graphs tab on the Results page allows you to create your own graphs of results data after running a simulation.

To show and hide graphs:

1.Click the Graphs tab at the top of the Results page.

2.Click Create Graph.

3.Choose a graph type, and type labels for the title and x- and y-axes.

4.In the list of variables, choose variables to include in the graph. Type a few letters of a variable name to filter the list.

5.You can adjust the width of graph lines, text style, legend position, and grid line style.
