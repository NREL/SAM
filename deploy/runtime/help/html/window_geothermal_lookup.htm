Geothermal Resource Lookup

The Geothermal Resource Lookup window allows you to download depth and temperature data for locations in the United States from an online database, and automatically set up a parametric simulation using the downloaded data.

Note. Your computer must be connected to the internet to use this feature.

Location Information

Type a street address zip code, or latitude and longitude. For example, any of the following will return results for the same location:

1617 Cole Boulevard, Golden CO

80401

39 44 N 105 09 W

39.75 -105.15

Query

SAM searches the online database for geothermal resource data available for the address or coordinates in Location Information:

•If data is available, SAM populates Latitude, Longitude, and the Depth and Temperature table with values.
•If data is not available, SAM populates Latitude and Longitude with values, and one row of the table with n/a.

Latitude and Longitude

The location's latitude and longitude, based on the address or coordinates in Location Information.

Depth (m) and Temp ('C)

A table of values from the online database for the temperature in degrees Celsius at different depths in meters below the earth's surface. The Depth vs Temp graph is a graphical representation of the data in the table.

Copy to Clipboard

Copies data from the depth and temperature table to your computer's clipboard.

Make Parametric Simulation

Creates a linked parametric simulation from the data in the Depth and Temperature table.

Close

Closes the Geothermal Resource Lookup window.
