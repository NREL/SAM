Technology and Market


Contents

The Technology and Market window allows you to choose options for a new case or modify options for the current case.

Select a technology

Choose a performance model for the case. For a description of the technology options, see Technology Options.

Select a financing option

Choose a financing model. For a description of the financing options, see Financing Overview.

The Technology and Market window appears:

●When you create a new project, see Start a Project for details.

●When you create a new case. See Manage Cases.

●When you click Select Technology and Market to modify the current case.

Note. Whenever you make changes in the Technology and Market window, SAM automatically populates the values of input variables with default values. Be sure to review the variables on each input page to make sure that they are appropriate for your analysis.

To choose technology and financing options:

1.Under Select a technology, click the technology and modeling option you want to use for your analysis.

2.When you click a technology option, SAM expands a list of modeling options for that technology, and displays the financing options available for that technology.

3.Under Select a financing option, choose a financing option.

4.If you are modifying options for a current case SAM displays a check box at the bottom of the window.

Check Reset new inputs to Tech/Market-specific default inputs to populate variables on the input pages with the default values. This will replace any values that may have been saved in previous model runs and is the default option.

Clear Reset new inputs to Tech/Market-specific default inputs to use values stored from previous model runs. This is useful if you are switching between technologies and want to revert to input values that you saved in an earlier analysis.

5.Click OK to return to the main window. SAM displays the technology and market option in brackets for your reference.
