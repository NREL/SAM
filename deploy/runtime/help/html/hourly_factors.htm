Curtailment and Availability

The Edit Hourly Factors window allows you to specify hourly factors that adjust the system's hourly energy value calculated by the performance model. You can use these factors to model a system's downtime for maintenance (availability) or shutdowns required by the system operator (curtailment). For a description of modeling applications for the hourly factors, see Degradation and Outages.

SAM multiplies the system's hourly electricity output by the factor you specify: A factor of one would result in no adjustment, a factor of 0.95 would reduce the output by 5%, and a factor of 1.05 would increase the output value by 5%.

Annual Factor

SAM multiplies the net electricity output value for each time step by the factor you specify, which effectively applies the factor to the system's total annual output.

For example, a value of 0.95 for a system with a net annual output of 100,000 kWh results in a delivered electrical output of 95,000 kWh.

Enable hourly factors

Check the box when you have a table of 8,760 adjustment factors. Click Edit data, and either cut and paste a column of 8,760 values into the table, or import a text file with a header row (column label) in the first row followed by 8,760 rows of factors. The first row of data applies to the hour beginning at 12 a.m. on January 1.

Enable hourly factors with custom periods

Check the box when you want to apply adjustment factors to specific time periods. Choose a month, date, and time for the start time and end time, and specify a factor for that period. Click Add period to specify more than one time period.
