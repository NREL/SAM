Case Compare

Case Compare is a flexible viewer and editor that allows you to explore the data in your SAM files. It allows you to build a table of input and results variables and their values and:

•Compare variables and values across cases in your file
•Edit values of input variables
•Export lists of variables and values

Note. Although you can use Case Compare to create tables of results, the tools on the Results page are better suited for that purpose.

To open Case Compare, on the Tools menu, click Case Compare.

Choose Variables to Display in the Table

The variables that appear in the table depend on the case, groups, inputs, and outputs, along with the display mode options that you choose.

To choose the case, groups, inputs, and outputs:

1.Under Total Cases, check the box for each case to include in the comparison or to export.
2.Under Total Groups, check the box for each group.
3.Review the list of variables under Inputs Selected. SAM generates the list based on the checked boxes under Total Cases and Total Groups.

You can add to or remove from variables in the list by clicking Add/Remove.

4.If you want to include results variables in your comparison or list to export, check the appropriate boxes under Total Outputs.

Total Cases

A list of all of the cases in the SAM file.

SAM displays the number of cases in the file. For example for a file with two cases, SAM displays 2 total cases.

Total Groups

A list of input variable groups available for the cases you choose under Total Cases. A group contains related input variables. For example, the Financing group contains input variables for the Financial model. A single input page in SAM may display variables from one or more groups.

SAM displays the number of available groups based on the cases you choose under Total Cases. For example, if there are 27 groups available, SAM displays 27 total groups.

Inputs Selected

A list of the input variables available to compare or export. The list depends on the check boxes under Total Cases and Total Groups.

SAM displays the number of available input variables for the combination of cases and groups you choose, and for variables you add or remove. For example, if there are 49 input variables, SAM displays 49 inputs selected.

Total Outputs

A list of results variables available to compare of export. The list depends on the check boxes under Total Cases and Total Groups.

SAM displays the number of results variables available for the combination of cases and groups you choose. For example, if there are 22 results variables available, SAM displays 22 total outputs.

Choose the Table's Display Mode

After you choose the variables to display in the table, you can use the options under Display to choose how SAM displays the variables. You can choose to display variables that exist in any or all cases, and variables that have the same or different values.

To choose the display mode:

1.Under Display, choose a Case mode to show variables that appear in any case or all cases.
2.Choose a Value mode to show variables with the same value, different values, or both to show all variables.
3.Check Hide N/A Rows to hide variables with the N/A value in all cases. Clear the box to show all variables with N/A values.

Case

The Case options determine the variables that appear in the table based whether or not they exist in the cases you choose under Total Cases. Only variables from the groups you choose under Total Groups appear in the table.

Any

Displays variables that exist in any of the cases you choose under Total Cases.

All

Displays only variables that exist in all of the cases you choose under Total Cases.

For example, in a file that contains a case with the residential financing option and another with the utility single owner financing option, Any would display the Financing group variables that apply to either the residential or the single owner option, and All would only display only the Financing group variables that apply to both financing options.

Value

The Value options determine the variables that appear in the table based on their values.

Same

Displays variables that have the same value in all cases.

Different

Displays variables that have different values in one or more cases.

Both

Displays all variables, regardless of their value, i.e., both variables that have the same value and different values.

Note. A yellow highlight in the table indicates variables that have different values.

N/A Rows

SAM displays N/A ("not applicable") in the table for variables that do not exist in a case. For example, if you compare a case for a photovoltaic system with a case for a wind system, and check the PV Module group under Total Groups, SAM displays "N/A" in the wind case's column for the module variables.

Hide N/A Rows

Check this box to hide rows that contain "N/A" in all columns of the table.

Numbers of Variables

SAM displays the number of variables for each combination of Case and Value modes for reference. These numbers do not change when you choose different display options.

Copy and Paste Values

You can use Case Compare to copy and paste data between SAM and a spreadsheet or other document, and to modify values of input variables in the cases.

Note. Use tab-delimited data to copy and paste values into the Case Compare table. You can copy and paste data directly between the Case Compare table and Excel.

Case Compare relies on your computer's clipboard to export data to spreadsheets and other documents.

Copy

Copies data from the Case Compare table to your computer's clipboard as tab-delimited data.

Paste

Pastes data from your computer's clipboard to the Case Compare table. The data in the clipboard should be tab-delimited with the same number of rows and columns as the table. Do not include row and column headers.

Copy Values from Cases to Table

Replaces values in the Case Compare table with values from each case checked under Total Cases.

Copy Values from Table To Cases

Replaces input variable values in each case checked under Total Cases with values from the Case Compare table.

To export the table to a spreadsheet or other document:

1.Choose the variables you want to appear in the table.
2.Click Copy to copy the table to your computer's clipboard as tab-delimited data.
3.Open the spreadsheet or other document.
4.Paste the table into the document.

To export a list of all input variables and their values from a single case:

1.Check the case name under Total Cases.
2.For Total Groups, click Select All.
3.To include results variables in the list, under Total Outputs, click Select All.
4.Under Display, click Any (case) and Both (value).
5.Click Hide N/A Rows.
6.Click Copy.
7.Open a spreadsheet or other document and paste the data into the document.
8.Click Close to exit Case Compare.
9.You can change values of variables in the table and copy them to cases so that they appear on the SAM input pages after you close Case Compare.

Change Values of Input Variables in your SAM File

One application of Case Compare is to verify and edit values of input variables in your cases. For example, if your file contains two cases with the same financing option, you can use Case Compare to ensure that the values of all of the financing input variables are the same for both cases.

Important Note! You cannot automatically undo the changes you make to variable values. Be careful to check your changes before copying values from the table to cases.

To update values on input pages to match those in the Case Compare table:

1.Choose the cases and variables you want to compare and edit.
2.Type new values for variables in the table.
3.Check the values to make sure they are correct. You cannot automatically undo changes after you copy values to the cases!
4.Click Copy Values from Table to Cases.
5.SAM lists the changes it made in the text box at the bottom right of the Case Compare window.
