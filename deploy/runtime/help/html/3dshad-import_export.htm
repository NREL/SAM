Import and Export

A .s3d file is a binary file that stores information describing the shading scene.

To import a .s3d file

1.Click Import to open the Load Scene from File window.

2.Navigate to the folder and click the file name.

To export a .s3d file

•Click Export to open the Save Scene to File window.
