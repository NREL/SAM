Debt Fraction

The debt fraction is the project debt amount as percentage of the total project cost. For example, a debt fraction of 25% means that the project borrows 25% of the total installed cost for a debt-equity ratio of 25%.

The debt fraction is an input on the Financial Parameters page for the following financial models:

•Residential
•Commercial

For the following PPA financial models that involve project debt, SAM calculates the project costs and debt and equity amounts to satisfy the assumptions you specify on the System Costs page and Financial Parameters page:

•Single Owner
•Leveraged Partnership Flip

For the single owner and leveraged partnership flip options, the debt fraction is:
