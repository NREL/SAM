Registration

When you start SAM for the first time, you must provide an email address to receive a registration key and activate your copy of SAM. Your registration key is associated with the email address you provide when you register your copy of SAM. The email address must be valid and for an account that you can access to ensure you receive the registration key.

•NREL provides SAM registration keys for free.
•The registration key works with both the Windows and OS X versions of SAM.
•You can use your registration key to register SAM on more than one computer.

Your personal data and SAM

After registering SAM, each time you start SAM, it verifies your registration key, and sends the following data to a secure NREL server, where it is stored permanently, and can be accessed by NREL staff:

•Your email address.
•Your computer's IP address with location codes (state, country, city), latitude, and longitude.
•The name of the SAM application and its version number.
•The date and time you registered each version of SAM.
•The date and time you last started each version of SAM.
•The number of times you started each version SAM.

NREL uses this data to determine how many people are using different versions of SAM, and approximately where people are using SAM. It includes this information in reports and documents for the U.S. Department of Energy to meet project documentation requirements and in project funding proposals.

Using SAM while offline

When you start SAM while your computer is not connected to the internet, the SAM registration window appears with an option to "skip for now." Use this option to start SAM while your computer is offline. You can run SAM offline, but you will not be able to download weather data, utility rate data, or other data from online databases. SAM allows you to run your copy of SAM 15 times without verifying the registration key.

Tip. If you plan to work offline during a trip or for another reason, keep SAM open on your computer to avoid using up the number of starts allowed without verifying your registration key. You can keep SAM open and shut down your computer.

 

Troubleshooting the registration process

SAM registration and the SAM Website

The SAM registration system is separate from the SAM website registration system. You do not need to register on the SAM website to download the current versions of SAM.

You do need to register on the SAM website to post questions and comments on the SAM support forum, or to download SAM 2014.1.14 and older versions of the software.
