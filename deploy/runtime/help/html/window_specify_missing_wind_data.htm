Specify Missing Wind Data


Contents

When you use the Download Weather File feature to download wind resource data for a location east of about longitude 100° west in the United States, SAM downloads data from the Eastern Wind Data set and stores it in your weather file folder in a file with the .srw format.

The Eastern Wind Dataset only contains wind speed data at 50 and 80 meters above the ground. It does not contain wind direction, ambient temperature, or atmospheric pressure data that SAM requires to simulate the wind power system.

The Specify Missing Wind Data window allows you to provide the missing data to make it possible to use the wind power model with the Eastern Dataset wind speed data.

After you specify the data and click OK, SAM downloads the data from the NREL database and saves a .srw file containing the data in your weather file folder.

Direction

Specify a monthly average direction value in degrees for each month of the year.

For example, if the prevailing wind direction in May, June, and July is West (the wind predominantly blows from the west during these months), then specify a value of 270° in the table for those months.

Yearly average temperature

The annual average ambient temperature at the site in degrees Fahrenheit.

Yearly average pressure

The annual average atmospheric pressure in millibars.

Collect wind speed information from year

Choose a year for the wind speed data. The Eastern Wind Dataset contains wind speed data for 2004, 2005, and 2006.

Note. SAM's wind power model uses the ambient temperature and atmospheric pressure in the wind power calculations. However, those parameters have a relatively small impact on the simulation results compared to the wind speed, so it is not critical to have accurate values.
