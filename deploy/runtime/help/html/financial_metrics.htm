Financial Metrics

SAM displays financial metrics on the Results page after running simulations.

For a general description of the Metrics table, see Metrics Table.

For a list of financial metrics with links to descriptions, see Financial Metrics Overview.

The financial metrics are:

•Project Costs
•Debt and Equity
•Debt Fraction
•Financing Cost
•Electricity Cost and Savings
•Internal Rate of Return (IRR)
•Land Area
•Levelized Cost of Energy (LCOE)
•Minimum DSCR
•Net Present Value (NPV)
•Payback Period
•PPA Price
•PPA Price Escalation
•Real Estate Value Added
