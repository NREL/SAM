Toggle Icons in PDF

To prevent toggle icon from appearing with Heading 2 text in PDF versions, use Text Crawler to make replacement in XML topic files:

•<image src="ico-plus-16x16.bmp" scale="100.00%" styleclass="Heading2"></image><tab />
•<conditional-text type="IFNOT" value="PDF,PRINT"/><image src="ico-plus-16x16.bmp" scale="100.00%" styleclass="Heading2"></image><tab /><conditional-text type="END"/>

After text has been replaced in a topic and I make changes in the topic, does H&M overwrite the topic or change incrementally? Need to check before each release.

Webhelp Publishing Options

Check the following Premium Pack (PP) options:

•PP 3-frame WebHelp
•PP Breadcrumbs
•PP Print button
•PP Toggles button

Do not check Enable local testing for MS Internet Explorer. Checking it causes the header file not to appear in the header panel in IE9. (9/1/11)

SAM Version Number

In MNL file:

•Change version number on title page
•Change version Number in all footers

Check that screenshot on title page is still valid

In Configuration, Common Properties, Text Variables (in project, not skin), change version variable value. Variable value should be version number only, e.g., 2011.6.30, because it appears as the file path name for the installation folder in some places.
