Financial Parameters

The Financial Parameters page is where you specify assumptions describing the project's financial and debt structure, tax rates, and other analysis parameters. SAM displays a different set of inputs depending on the financial model.

The inputs on the Financial Parameters page are different for the different financial models:

•Residential

•Commercial

•Third Party Ownership - Host

•Third Party Ownership - Host / Developer

•Single Owner

•Merchant Plant

•Leveraged Partnership Flip

•All Equity Partnership Flip

•Sale Leaseback

•LCOE Calculator
