Annual Schedules

SAM uses a table of values called an annual schedule for variables whose value changes from year to year as defined in an annual schedule table. Examples of input variables with an annual schedule option are:

•Operation and maintenance costs on the System Costs page
•Escalation rate for electricity prices on the Electricity Rates page
•Annual degradation rate on the Degradation page
•Production tax credit and production-based incentive on the Incentives page

Annual Schedules and Inflation or Escalation Rates

In some cases, variables with an annual schedule option are associated with the inflation rate (from the Financial parameters page) or an escalation rate. For those variables, the escalation rate applies to the single value, but does not apply when you use an annual schedule. For example, for the production-based incentive on the Incentives page, the escalation rate for each PBI (federal, state, etc.) applies when you specify the incentive as a single value, but not as a table of values.

Tip. When you first start working with annual schedules, it is a good idea to try some test values and see how they affect results to be sure you understand how SAM interprets the values.

Important Note! When you specify annual rates using an annual schedule table, SAM applies the rate in each year to the first value in the table.
 
For example, on the Degradation page, an Annual decline in output value of 0.5% for Year 5 would when the Year 1 net annual output is 10,000 kWh would result in a delivered annual output value of (1 - 0.005) × 10,000 kWh = 9,950 kWh in year 5.
 
Similarly, on the Electricity Rates page, an for a Year 1 buy rate of $0.10/kWh and an Out-year escalation value of 0.25% would result in a Year 3 buy rate of (1 + 0.0025) × $0.10/kWh = $0.10025/kWh.

To enter an annual schedule:

1.Click Sched to change the variable's mode from a single value to annual schedule mode. SAM displays the Edit button.

2.Click Edit to open the Edit Schedule window. Note the variable's units on the SAM input page because the Edit Schedule window does not show the units.

3.In # of Values, type the number of years for which you want to assign values. Typically, this number should be equal to or less than the number of years in the Analysis Period defined on the Financing page.

Important Note. If you specify a number greater than the number of years in the analysis period, SAM ignores any values in the table for years after the end of the analysis period, which does not affect analysis results. However, if you specify a number less than the analysis period, SAM assigns a zero to each year after the number of years you specify, which may cause unexpected results.

4.For each year in the schedule, type a value.

You can also copy a row of values from Excel, or a line of comma separated values from a text file to your computer's clipboard, and click Paste to paste them into the table.

5.Click OK to save the values and return to the SAM input window.
