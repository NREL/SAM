Software Development Kit (SDK)

The SAM Simulation Core (SSC) software development kit is a collection of developer tools for creating renewable energy system models using the SSC library. SAM is a desktop application that provides a user-friendly front end for the SSC library. The SDK allows you to create your own applications using the SSC library.

To download the SSC SDK, visit the SAM website SDK page.

Note. The SSC SDK is under development at the time of this SAM release (mid-January 2013). It replaces the Code Generation feature that was part of earlier SAM versions.

The SDK includes the following:

•The SSC Guide explaining how to use the tools in the SDK.
•The SSC API.
•The SSCdev development environment to explore SSC modules and build test models.

A set of libraries for the following operating systems:

•Windows 32-bit
•Windows 64-bit
•OS X 64-bit
•Linux 64-bit

A set of wrappers for the following languages:

•C#
•Java
•MATLAB
•Python
