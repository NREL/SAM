Edit / Add New Graph

The Edit Graph and Add New Graph windows allow you to specify the properties of a graph. As you change properties, SAM automatically updates the graph on the Results page so that you can see the effect of the changes as you work.

Note. SAM's graphing tool is intended to be flexible enough to generate a wide variety of graphs, but may not be suitable for all graphing applications. You can always export graph data to use with your own graphing software.

To change the properties of an existing graph:

1.Display the Results page.
2.Click Graphs.
3.Double-click the graph's thumbnail at the bottom of the page,
4.Select the graph and then click Edit, or right-click the graph (Windows only).
5.Change the graph's properties in the Edit Graph window.
6.Click Accept to save the changes.

To create a new graph:

1.Display the Results page.
2.Click Graphs.
3.Click Add a new graph.
4.Specify the properties of the new graph in the Edit Graph window.
5.Click Accept to create the graph.

Graph Data

The data available for graphing depends on the technology and financing options, other model options (such as the photovoltaic module model option), and the analysis options you include in your analysis. SAM displays different variables in the lists of Y1 values, Y2 values, and Z values depending on the options you choose for your simulations. For descriptions of the variables see:

•Time series (hourly) and monthly performance data: Performance model outputs.
•Annual cash flow data: Cash flow variables
•Time-dependent pricing data: Savings and Revenue

Choose Simulation

Choose the simulation on which you want to base the graph.

The base case option is always available and displays variables for the base case results, which are results calculated using input values on the input pages.

Other options are available when you specify additional analysis options. For example, if your analysis includes parametric variables, to create graphs of parametric results, choose the Parametric Set option. Similarly, you can choose the appropriate option when your simulations include sensitivity, statistical, or P50/P90 analysis results.

X Value

Variable to display on the horizontal axis. Applies to Bar, Stacked Bar, Line Plot, Histogram/CDF, and Tornado Chart.

All Years displays values calculated for each year of the analysis period, and is useful for plotting cash flow data.

Lifetime (Monthly) applies only to the Geothermal model, which is still under development.

Monthly displays monthly average values.

Single Value displays a single point on the graph,such as the LCOE, and works best with bar graphs.

Y1 Values

Variable to display on the left-hand vertical axis. Applies to Bar, Stacked Bar, Line Plot, and Tornado Chart. Check a box for each value you want to appear on the graph. You can adjust the axis scale and colors used to display the values using the Properties settings described below.

Y2 Values

Variable to display on the right-hand vertical axis. Applies to Bar, Stacked Bar, and Line Plot. Check a box for each value you want to appear on the graph. You can adjust the axis scale and colors used to display the values using the Properties settings described below.

Z Value

Variable to display as the independent variable on a contour plot. Contour plots are only available for parametric analyses involving at least two parametric variables.

X Parametric

Parametric variable to display on the x-axis of a contour plot. Variable names only appear when one or more parametric variable is defined. Applies to Contour Plot and requires a parametric set with at least two parametric variables.

Y Parametric

Second parametric variable to display on the y-axis of a contour plot. Variable names only appear in list when two or more parametric variables are defined. Applies to Contour Plot and requires a parametric set with at least two parametric variables.

Graph Type

Bar Graph

Represents a y value for each x value as a vertical rectangular bar. To create a bar graph, choose a simulation to graph (base case, parametric etc), and for X Value, choose All Years, Monthly, or Single Value, then check a box under Y1 Values for each value you want represented by a bar.

Stacked Bar Graph

Similar to a bar graph, but rectangles within each bar represent categories of data. To create a stacked bar graph, choose a simulation to graph (base case, parametric etc), and for X Value, choose All Years, Monthly, or Single Value, then check a box under for Y1 Values for each value you want included in the bar. Checking boxes for both Y1 and Y2 creates a set of two stacked bar graphs.

Line Plot

A line connects y values on a two-dimensional graph. To create a line graph, choose a simulation to graph (base case, parametric, etc.), and for X Value, choose an option or parametric variable.

Tornado Chart

Horizontal rectangular bars represent the variation in the value of an output variable with respect to one or more input variables. Available for sensitivity simulations.

Contour Plot

A two-dimensional representation of a three-dimensional surface. Available for parametric simulations with two or more parametric variables.

Histogram/CDF

Represents the frequency of occurrence of an output variable over a range of input variable values. Available for statistical simulations.

Scatter Plot

Available for statistical simulations.

Properties

Title

The title that appears at the top of the graph.

X Label

Axis label for the horizontal axis.

Y1, Y2 Label

Axis labels for the left-hand vertical axis (Y1) and right-hand vertical axis (Y2)

Show X, Y1, Y2 Values

Use these check boxes to show and hide data on the graph.

Text Size

Control the font size of graph titles, axis labels, and numbers on the graph. SAM displays all text and number using the same font size. Zero is the default size, positive values are for larger text, and negative values for smaller text.

X Tick Spacing

Control how tick marks on the x axis are labeled. For example, a value of one places a numeric label on each tick mark. A value of five places a numeric label at every fifth tick marks.

Thickness

Control the thickness of lines and bars on the graph.

Show Legend

Show or hide the graph legend.

Show Coarse Grid

Show or hide widely spaced grid lines on the graph.

Show Fine Grid

Show or hide narrowly spaced grid lines on the graph.

Set X, Y1, Y2 Min/Max

Specify the minimum and maximum values for the x axis and two y axes.

Color Schemes

Change the colors used to display lines and graphs. You can only choose different color schemes. You cannot assign specific colors to individual graph elements.
