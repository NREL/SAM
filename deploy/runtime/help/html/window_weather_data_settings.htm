Weather Data Settings


Contents

The Weather Data Settings window allows you to choose weather file folders where SAM looks for weather files, and where it saves weather files that you download from the web with the Download Weather File feature.

Weather File Folders

The list of folders on your computer that SAM searches for weather files.

SAM lists any files with an extension that it recognizes as a weather file (.tm2, .csv, .srw, .smw, .epw) in the list of weather files on weather file input page.

Add

Add a folder to the list of weather file folders.

Remove

Click a folder in the weather file folder list and click Remove to remove the folder.

Folder for Downloaded Weather Files

When you use the Download Weather File feauture to download weather data from an online database, SAM stores the weather file in the folder that you specify.

The folder can be the same as a folder listed under Weather File Folders, or a different folder.
