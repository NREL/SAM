Summary

The Metrics table on the Results page displays a selected set of results variables for each case in the project file.

For a list and descriptions of the variables in the Metrics table, see

•Financial Metrics for metrics from the financial models, such as LCOE, payback period, IRR, etc..
•Performance Metrics for metrics from the performance models, such as annual energy, capacity factor, etc.

SAM displays the Metrics table under the navigation menu when results are available for a case. The variables that appear in the metrics table depend on the technology and financial models.

Note. Values like sv.npv or [not found] indicate that there are no results to display. Try running a simulation to display the values.
