Graphs

SAM displays graphs on the Results page after running a simulation. The standard graphs that appear by default show summaries of results from both the performance model and financial model. You can edit the standard graphs and create your own graphs.

Note. Values like sv.npv or [not found] indicate that there are no results to display. Try running a simulation to display the values.

To show and hide graphs:

1.Click Graphs at the top of the Results page.

2.Click Create Graph.

3.In the list of variables, choose variables to include in the graph.

4.Choose a graph type, and type labels for the title and x- and y-axes.

5.You can adjust the width of graph lines, text style, legend position, and grid line style.
