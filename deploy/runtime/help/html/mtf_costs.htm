Project Costs

For the advanced utility IPP financing options, SAM displays some key cost metrics on the Results page for convenience in assessing the financial viability of a project:

•Single owner
•Leveraged Partnership Flip
•All Equity Partnership Flip
•Sale Leaseback

The direct and indirect costs are inputs that you specify on the System Costs page.

Direct Cost

The sum of the direct installation costs from the System Costs page.

Indirect Cost

The sum of indirect installation costs from the System Costs page.

Total project cost

The total project cost is the sum of direct, indirect and financing costs:

Total Project Cost ($) = Direct Cost ($) + Indirect Cost ($) + Financing Cost ($)
