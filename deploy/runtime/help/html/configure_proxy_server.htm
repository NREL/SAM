If your organization uses a web proxy server to connect to the internet, and you are having trouble accessing online databases from SAM, you may need to create a proxy configuration file. The file is a one-line text file with one proxy server address and an optional custom port.

To create a proxy configuration file:

1.If SAM is open, save your work, and close it.
2.Open the SAM installation folder (it contains the sam.exe executable file).
3.Use a text editor to create a new empty text file named proxy.txt, and save it in the installation folder.
4.Put one line in the text file with your organization's web proxy server's address on a single line:

proxy-server.myorganization.org

To specify a custom port, add it to the name with a colon separator (no spaces). For example, if the custom port number is 9142:

proxy-server.myorganization.org:9142

5.Save the file.
6.Close SAM.

The next time you start SAM, it will use the proxy server address to access the internet.
