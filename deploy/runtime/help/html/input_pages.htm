Input Pages

Use the input pages to verify and change values of input variables for your analysis. Use the navigation menu to display different input pages. SAM displays different kinds of variables in different colors.

SAM's input pages provide access to the input variables and options that define the assumptions of your analysis.

When you start a project by creating a new file SAM populates all of the input variables with default values so that you can get started with your analysis even before you have final values for all of the input variables.

Tip. To see a list of all input variables and their values for each case in your file, on the main menu, click Browse case inputs .
