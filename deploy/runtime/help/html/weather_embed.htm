Embed a Weather File

When you want to share a SAM project with another person, and the project uses one or more weather files that the other person does not have, you can include a copy of the data from the weather files in the SAM file. Embedding a weather file increases the size of the SAM file, but also makes it more portable.

To copy data from a weather file to the project file:

1.Choose the weather file from the Location list.
2.Click Copy to project.

SAM adds the file to the location list with the "USER/" prefix, indicating that the data is included in the SAM project file.

To remove an embedded file, select it from the list, and click Remove from project.
