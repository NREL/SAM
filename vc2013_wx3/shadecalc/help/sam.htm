Use shade data in SAM

The System Advisor Model (SAM) photovoltaic models (Flat Plate PV and PVWatts System) can import diurnal and hourly shade data from files that you export from the shade tool. Import the data from SAM's Edit Shading Data window:

•In the Flat Plate PV model, on the PV Subarrays page, click Edit Shading.
•In the PVWatts System model, on the PVWatts Solar Array page, click Edit Shading.

Keep the following things in mind:

•The shade data only works in SAM for fixed arrays and subarrays. It does not work with any of the tracking options.
•If your system consists of multiple subarrays, you can represent each of the subarrays in SAM with a separate active surface in the shade calculator by giving each active surface a different group name.
•If the system in SAM has a single subarray, you can use either diurnal shade data (hour by month, 288 values) or hourly data (8,760 values) from the shade tool.
•If the system has multiple subarrays, you can only use diurnal shade data if you want separate shade data for each subarray. The calculator generates hourly data for the entire array, but not for each active suface.
•Use the Beam shading factor option with SAM. (Do not use the shading fraction data.)
•The shade calculator tool does not generate a sky diffuse shading factor.

When you create a case in SAM to use with shade calculator data, verify the following:

•The latitude, longitude, and time zone on SAM's Location and Resource page match the location in the calculator.
•The array tilt and azimuth angles in SAM match the active surface tilt and azimuth angles in the calculator.

To use diurnal shade data

1.Open the scene file in the shade tool, and the project file in SAM. Keep both the shade tool and SAM open.
2.In the shade tool:
a.Click Analyze, and generate the shade data.
b.For a single SAM subarray, click Export under Entire Array, and save the CSV file.
c.For multiple SAM subarrays (Flat Plate PV only), click Export under each active surface's hour-by-month table, and save the CSV files. Use file names that will help you identify which file belongs to which subarray in SAM.
3.In SAM:
a.Open the Edit Shading Data window as described at the top of this page.
b.Check Enable month by hour beam irradiance shading factors.
c.Click Import under the hour-by-month table, and open the CSV file.
d.For multiple subarrays repeat Steps a-c to import data for each subarray.

To use hourly shade data (single subarray only)

1.Open the scene file in the shade tool, and the project file in SAM. Keep both the shade tool and SAM open.
2.In the shade tool:
a.Click Analyze, and generate the shade data.
b.Click Hourly analysis (entire array), and save the CSV file.
3.Open the CSV file in a spreadsheet program, and copy the Shading Derate values (second column) in Rows 2-8761 to your computer's clipboard. Do not include the header row (Row 1 with the column label).
4.In SAM:
a.Open the Edit Shading Data window as described at the top of this page.
b.Check Enable hourly beam irradiance shading factors.
c.Click Edit data.
d.Click Paste under the hour-by-month table, and open the CSV file.

Important note! If you try to import the CSV file instead of copying and pasting, SAM will import the first column, which contains the shade fraction values instead of the beam shade factors that SAM requires.
