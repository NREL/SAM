Open and Save

A .s3d file is a binary file that stores information describing the shading scene.

To open a .s3d file

1.Click Open to open the Load Scene from File window.
2.Navigate to the folder and click the file name.

Note. You can also open a .s3d file by clicking its name in your file browser.

To save a .s3d file

1.Click Save to open the Save Scene to File window.
oTo overwrite an existing file, click Save.
oTo save the file with a different name, type a file name, and then click Save.
